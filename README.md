# DSE_News_Scraper_with_Automated_LR_16-19_Classification
This Python script efficiently scrapes all news published on the Dhaka Stock Exchange (DSE) within a user-defined date range. It automatically extracts, classifies, and organizes LR 16(1) and LR 19(1) news

##Key Functionalities:
-Scrapes the complete news archive from DSE for any user-defined date range.
-Automatically detects and separates LR 16(1) and LR 19(1) regulatory announcements.
-Cleans, structures, and formats all extracted news data.
-Exports results into an Excel file with organized worksheets, including.
  -All News.
  -LR 16(1).
  -LR 19(1).
-Ensures stable performance even when news items differ in layout or formatting.
